# PURPOSE

This template defines the standard structure for all **Galaxy concept notes**.

Each note represents a **single concept node** inside the VANTIS knowledge graph.

The template ensures:

- consistent metadata
- predictable structure
- compatibility with retrieval systems
- easy AI reasoning
